# hirop

A Clojure(Script) data layer targeted at multi-document consistency and asynchronous updates.

## Usage

Leiningen:
```clojure
[hirop "0.1.0-SNAPSHOT"]
```

Test:
```clojure
lein test
```

Please refer to the [hirop wiki](https://github.com/orobix/hirop/wiki).
